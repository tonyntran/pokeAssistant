# TCGPlayer Integration

## Overview

TCGPlayer is the primary data source for Pokemon TCG card pricing.

## Data Points to Track

- Market price
- Low / Mid / High prices
- Card condition (Near Mint, Lightly Played, etc.)
- Edition (1st Edition, Unlimited, etc.)
- Set name and card number
- Price history over time
